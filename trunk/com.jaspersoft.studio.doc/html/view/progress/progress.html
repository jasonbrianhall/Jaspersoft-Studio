Progress View

Jaspersoft Studio run all long operations like report execution in the background. Progress of this tasks is shown in the right part of the statusbar. In the statusbar we see the name of the current task, progress bar and a button that opens a view called Progress View from which we can see all the running jobs and if possible cancel them.

Sometimes it's possible to see a modal dialog during task execution, in this case user have to wait until task is finished, or push Run In Background button, that will hide this dialog. There is a Always run in background checkbox that will run all tasks in the background in the future.
