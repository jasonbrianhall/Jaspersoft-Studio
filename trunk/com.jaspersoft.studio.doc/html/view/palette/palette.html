Palette View

This view contains all graphic elements we can drag from palette to the graphic editor. There are two modes for this view, one inside report designer another as a separate view. To open palette in a separate view go to Windows → Show View → General → Palette if you close this view palette will be available inside the graphic editor.

It's possible to configure palette view just right clicking inside it, a context menu will propose options for big or small icons, the way elements are arranged (column, list, icons only, details), the place where palette is docked.

There are three ways to create elements from a palette:

  • drag and drop element over a report container like a band, when object is dropped this will be the position for the element

  • click on an element inside the palette, go to a report container like a band click in the place where you want the top left corner will be, and drag in right-down direction

  • drag from palette to outline view
