Properties View

Properties view help configuring a selected element or elements. In case more than one element is selected, only properties commont to all the selected elements are shown. If Properties view is not visible, look in the context menu of selected elements for Show Properties menu item, this one will show and bring to front the properties view.

Properties are organized in tabs, and sections for a better organization and usability. Usually we can see an Appearance ; tab, that contain all the properties responsible for element size, position, colors, style and flags when the element will be shown in the runtime. Another common tab is named Borders , here we can configure how borders of the element are painted. There is a special Advanced tab that contains all the properties for the report element in a form of a table tree, it's possible to reset the property to default value, in context menu.

For focused property a property description is shown in the tool tip and in the status line.
